# Ignition (formerly Practice Ignition) — Competitor Analysis

Market leader for accounting practice proposals, engagement letters, and billing automation.
Website: https://ignitionapp.com

## Core Workflow

### Phase 1 — Lead Capture (Deals)
- Built-in lightweight CRM with customizable pipeline (New, Qualifying, Negotiating, Closed)
- Branded intake forms auto-create deals
- Pipeline values update in real-time as clients sign or pay

### Phase 2 — Proposal Creation
- Built from a **Service Library** (reusable catalogue with standardised descriptions, pricing, billing frequencies)
- Services can be one-time, recurring, or $0 placeholders (scope boundaries)
- **Tiered pricing** (Essentials/Standard/Premium) within a single proposal
- Branded with firm logo, colours, and custom domains (Pro+)
- AI Price Insights ($349/year add-on) benchmarks pricing against anonymised platform data

### Phase 3 — Engagement Letters
- Embedded within proposals (not a separate document)
- **Dynamically composed** based on selected services
- Terms Library with customisable, versioned templates
- Legally vetted (ACCA-endorsed in UK/ANZ)
- Up to 10 signatories per engagement letter

### Phase 4 — Sending & Tracking
- Sent via email as a link to branded online page
- Opens tracked (firm sees when prospect viewed)
- Client can leave comments/questions directly on the proposal
- Bulk sending (Core plan+)

### Phase 5 — Acceptance
- Client reviews services, pricing, and terms on ONE page
- Electronic signature + payment details (card, direct debit, Apple Pay, Google Pay) in same flow
- Firm can "accept on behalf" if needed

### Phase 6 — Post-Acceptance Automation
- Invoices created in Xero/QBO
- Jobs created in Xero Practice Manager / Karbon
- Client records synced
- Payment collection begins automatically
- Team members notified/assigned

## Key Design Insight

Ignition **collapses proposal + engagement letter + payment authorisation into a single document/acceptance event**. No separate steps. The moment the client signs, they've simultaneously accepted scope, agreed to legal terms, and authorised payment collection.

## Billing Models

| Model | Description |
|-------|-------------|
| Recurring fixed fee | Monthly/quarterly/annual automatic charges (dominant model) |
| One-time fee | Single charge for discrete services |
| Hourly billing | Time-based billing |
| Upfront deposit | Collected at proposal acceptance |
| Milestone / on-completion | Invoice upon completion of deliverables |

Per-line billing frequency is configurable. A single proposal can mix models.

## Scope Creep Management

- **Instant Bill** — bill for ad-hoc/out-of-scope work immediately without a new proposal
- **Service Edits** — modify active engagements mid-term without re-engagement
- **$0 scope boundaries** — placeholder services that define what's NOT included

## AutoCollect (May 2025)

Imports unpaid invoices from Xero/QBO into Ignition. Invites clients in bulk to pay via online portal. Pre-saved payment methods for auto-collection. Turns Ignition into a collections tool for ALL invoices.

## Xero Integration

| Capability | How It Works |
|------------|-------------|
| Invoice creation | Accepted proposals auto-generate Xero invoices |
| Client matching | Name match or auto-create new contact |
| Payment reconciliation | Ignition payment → Xero invoice marked paid + reconciled |
| Revenue account mapping | Per-service chart of accounts code |
| Tracking categories | Assigned at client level, auto-applied |
| Invoice import | Import existing recurring Xero invoices as proposals |
| Unpaid invoice import | AutoCollect pulls 90 days of unpaid invoices |

## Key Differentiating Features

1. Single-acceptance flow (proposal + terms + payment auth)
2. Automatic payment collection (91% auto-collected)
3. Tiered pricing in proposals (encourages upselling)
4. Dynamic engagement letter composition
5. AI Price Insights (benchmarking)
6. Bulk renewals with automatic price increases
7. Scope creep tools (instant bill, service edits, $0 boundaries)
8. Branded client experience (custom domains, landing pages)

## Our Advantage Over Ignition

Ignition is billing-only. It pushes everything to Xero for the actual accounting. MoneyQuest Ledger has the full accounting engine:
- Proposals link to jobs which link to journal entries and chart accounts
- Real profitability: proposal price vs actual costs (from posted ledger entries)
- Time tracking (072-JTW) connects effort to billing
- No Xero dependency — the ledger IS the financial system
