## Phase 3 — Graph Data Endpoints

- [ ] T018 Action: Create `GetGraphData` — use AsAction. handle(User $user, ?int $groupId = null): array. Steps: (1) get accessible workspace IDs via $user->workspaces()->pluck('workspaces.id'), (2) if groupId, filter to group members, (3) load workspaces with id/name/entity_type/slug/base_currency/created_at, (4) compute net worth per workspace from AccountBalance (sum assets - sum liabilities), (5) load entity_relationships where both source and target are in accessible set (include soft-deleted=false), (6) load ghost nodes (workspaces outside accessible set that have relationships to accessible workspaces — return id/name/entity_type/is_ghost=true only), (7) load WorkspaceGroups containing any accessible workspace with member IDs, (8) load user's graph_node_positions. Return {nodes, edges, groups, positions, ghost_nodes}. File: app/Actions/Graph/GetGraphData.php
- [ ] T019 [P] Action: Create `GetNodeDetail` — use AsAction. handle(User $user, Workspace $workspace): array. Return: name, entity_type, base_currency, net_position (assets - liabilities from AccountBalance), contact_count, open_invoice_count, active_job_count, health_score (from HealthScore model if exists), relationships_outbound (from entity_relationships where source = workspace, with type label), relationships_inbound (from entity_relationships where target = workspace, with inverse label), recent_activity_count (from ActivityItem). File: app/Actions/Graph/GetNodeDetail.php
- [ ] T020 [P] Action: Create `SaveNodePositions` — use AsAction. handle(User $user, array $positions): void. Upsert graph_node_positions rows: foreach position, updateOrCreate by (user_id, workspace_id) with (x, y, updated_at=now()). File: app/Actions/Graph/SaveNodePositions.php
- [ ] T021 [P] Action: Create `GetHistoricalNetWorth` — use AsAction. handle(array $workspaceIds, Carbon $asOfDate): array. Query journal_entry_lines joined with journal_entries where entry_date <= $asOfDate and workspace_id in $workspaceIds. Group by workspace_id and chart_account.type (asset vs liability). Sum amounts. Return [workspace_id => net_worth_cents]. File: app/Actions/Graph/GetHistoricalNetWorth.php
- [ ] T022 Controller: Create `GraphController` — data(Request): call GetGraphData with user and optional group_id query param, return JSON. nodeDetail(Request, Workspace): verify user has access via workspaces pivot, call GetNodeDetail, return JSON. savePositions(Request): validate positions array (workspace_id, x, y), call SaveNodePositions, return 204. historicalNetWorth(Request): validate workspace_ids array and as_of_date, call GetHistoricalNetWorth, return JSON. File: app/Http/Controllers/Api/GraphController.php
- [ ] T023 Routes: Add graph routes to routes/api.php under auth:sanctum middleware (NO workspace context middleware). Routes: GET /graph/data → GraphController@data, GET /graph/node/{workspace} → GraphController@nodeDetail, POST /graph/positions → GraphController@savePositions, GET /graph/historical-net-worth → GraphController@historicalNetWorth, POST /entity-relationships → EntityRelationshipController@store, PATCH /entity-relationships/{entityRelationship} → EntityRelationshipController@update, DELETE /entity-relationships/{entityRelationship} → EntityRelationshipController@destroy, GET /entity-relationships/{entityRelationship}/history → EntityRelationshipController@history.
