## Phase 2: Pipeline Processing — Actions, Jobs, Scheduler

- [ ] T022 Action: `SyncFeedSource` — AsAction. `handle(FeedSource $source): SyncResult`. Resolve provider via FeedProviderRegistry. Call `$provider->fetch($source, $source->last_synced_at)`. For each raw item: call `$provider->normalise($rawItem)`, then upsert FeedItem (skip if external_id already exists for this source — idempotent). Update `$source->last_synced_at = now()`, `$source->next_sync_at = now()->addMinutes($source->sync_interval_minutes)`, `$source->error_count = 0`. On exception: increment `$source->error_count`, set `$source->error_message`, if error_count >= 3 set status to Error. For bank_transaction type: also create BankTransaction via existing ImportBankTransactions and link via bank_transaction_id. Return SyncResult DTO with counts (imported, skipped, total, errors). File: `app/Actions/Feed/SyncFeedSource.php`
- [ ] T023 [P] DTO: `SyncResult` — Spatie Data class: `int $imported`, `int $skipped`, `int $total`, `int $errors`, `?string $error_message`. File: `app/Data/Feed/SyncResult.php`
- [ ] T024 Action: `ClassifyFeedItem` — AsAction. `handle(FeedItem $item): FeedClassification`. Load active FeedRules for this item's workspace and type, ordered by priority ASC. For each rule, call `$rule->matches($item->normalised_payload)`. First match returns FeedClassification with matched=true, rule details, confidence=85 (rule-based default). If no match, return FeedClassification with matched=false, suggestions=[] (AI suggestions deferred to later epic). Update item status to Classified if matched. File: `app/Actions/Feed/ClassifyFeedItem.php`
- [ ] T025 Action: `TransformFeedItem` — AsAction. `handle(FeedItem $item, FeedClassification $classification): array`. Returns array of JournalEntryLineData-compatible arrays. For bank_transaction type: mirror existing ReconcileTransaction logic — bank side (debit/credit the bank chart account from the FeedSource's linked BankAccount) + contra side (the classified target_chart_account_id). Amount from normalised_payload. Tax code from classification. Return array of line arrays. File: `app/Actions/Feed/TransformFeedItem.php`
- [ ] T026 Action: `GenerateFeedJournalEntry` — AsAction. `handle(FeedItem $item, array $lines, FeedClassification $classification): JournalEntry`. Uses existing `CreateJournalEntry::run()` with workspace_id, date from normalised_payload, description from normalised_payload, lines array, source_type='feed', feed_item_id=$item->id. If classification.auto_post is true, also call `SubmitJournalEntry::run()` to post it. Return the JournalEntry. File: `app/Actions/Feed/GenerateFeedJournalEntry.php`
- [ ] T027 Action: `ProcessFeedItem` — AsAction (orchestrator). `handle(FeedItem $item): ProcessResult`. Call ClassifyFeedItem::run($item). If not matched → return ProcessResult::needsReview(). Call TransformFeedItem::run($item, $classification). Call GenerateFeedJournalEntry::run($item, $lines, $classification). Update item: status=Posted, journal_entry_id, classification json, processed_at=now(). Dispatch FeedItemProcessed event. Catch exceptions: increment retry_count, set error_message, if retry_count >= 3 set status=Error. Return ProcessResult. File: `app/Actions/Feed/ProcessFeedItem.php`
- [ ] T028 Action: `ProcessFeedBatch` — AsAction. `handle(FeedSource $source): array`. Load all FeedItems where feed_source_id=$source->id and status=Pending. For each item, dispatch ProcessFeedItemJob. Return count of dispatched jobs. File: `app/Actions/Feed/ProcessFeedBatch.php`
- [ ] T029 Job: `SyncFeedSourceJob` — Queue job. Constructor takes FeedSource $source. `handle()`: set workspace context via `setPermissionsTeamId($source->workspace_id)`, call `SyncFeedSource::run($source)`, then call `ProcessFeedBatch::run($source)`. Retries: 3, backoff: [60, 300, 900]. File: `app/Jobs/SyncFeedSourceJob.php`
- [ ] T030 Job: `ProcessFeedItemJob` — Queue job. Constructor takes FeedItem $item. `handle()`: set workspace context, call `ProcessFeedItem::run($item)`. Retries: 3, backoff: [30, 120, 600]. File: `app/Jobs/ProcessFeedItemJob.php`
- [ ] T031 Event: `FeedItemProcessed` — Stored event (Spatie). Properties: string $feed_item_uuid, string $feed_source_uuid, string $workspace_id, string $type, array $classification, ?int $journal_entry_id. File: `app/Events/Feed/FeedItemProcessed.php`
- [ ] T032 Command: `SyncDueFeeds` — Artisan command `feed:sync`. Loads FeedSource::dueForSync(). For each, dispatches SyncFeedSourceJob. Logs count of sources synced. File: `app/Console/Commands/SyncDueFeeds.php`
- [ ] T033 Schedule: Add `feed:sync` to `routes/console.php` — `Schedule::command('feed:sync')->everyMinute()`. File: `routes/console.php`
