## Key Differences from Jira

| Jira | Linear | Notes |
|------|--------|-------|
| Story, Task, Bug, Epic... | Just "Issue" | Simpler, use labels for type |
| Jira Plan timeline | Built-in timeline | Drag to set dates, cleaner UI |
| Sprint | Cycle | Same concept, 2-week default |
| Phases | Milestones | Planning → Design → Dev → QA → Release |
