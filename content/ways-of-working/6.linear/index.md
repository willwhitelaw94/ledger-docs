> Linear is our project management tool—simpler than Jira, built for modern development workflows.

---

## Why Linear?

Linear is designed for the way modern teams actually work:

- **Simpler hierarchy** — Issues, Projects, Initiatives (not 15 issue types)
- **Faster UI** — Keyboard-first, no page refreshes
- **AI-native** — Built-in AI features, great MCP integration with Claude
- **GitHub integration** — Branch linking just works

---

## Key Concepts

| Linear | Jira Equivalent | Description |
|--------|-----------------|-------------|
| **Initiative** | Epic (top level) | Major work streams |
| **Project** | Epic | Specific deliverables |
| **Issue** | Story/Task/Bug | All work items in one type |
| **Sub-issue** | Sub-task | Breakdown of an issue |
| **Cycle** | Sprint | Time-boxed iteration |
| **Milestone** | Phase | Planning, Design, Dev, QA, Release |
