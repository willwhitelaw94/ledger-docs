[Laravel Forge](https://forge.laravel.com/) is a server management and deployment platform designed for Laravel applications.

---

## What It Does

Forge handles server provisioning, deployment, and management for PHP applications.

| Feature | Description |
|---------|-------------|
| **Server Provisioning** | Automated setup on AWS, DigitalOcean, etc. |
| **Deployments** | Push-to-deploy from Git repositories |
| **SSL Certificates** | Free Let's Encrypt certificates |
| **Queue Workers** | Manage Horizon and queue processes |
| **Scheduling** | Cron job management |
| **Database Management** | MySQL/PostgreSQL provisioning |

---

## Our Usage

We use Forge for:

- Production and staging server management
- Automated deployments from GitHub
- SSL certificate management
- Queue worker and scheduler management
