[Laravel Nova](https://nova.laravel.com/) is a beautifully designed administration panel for Laravel applications.

---

## What It Does

Nova provides a powerful admin interface for managing application data, built specifically for Laravel.

| Feature | Description |
|---------|-------------|
| **Resource Management** | CRUD operations for Eloquent models |
| **Custom Fields** | Rich field types for forms and displays |
| **Actions** | Bulk operations on resources |
| **Filters** | Powerful filtering and search |
| **Metrics** | Dashboard cards and visualizations |
| **Authorization** | Integrates with Laravel policies |

---

## Our Usage

We use Nova for:

- Internal administration and data management
- Support team tooling
- Quick data lookups and corrections
- Reporting and metrics dashboards

---

## Access

Nova is available at `/nova` for authorized staff members.
