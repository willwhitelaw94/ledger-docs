[Laravel Herd](https://herd.laravel.com/) is a blazing fast, native macOS development environment for Laravel.

---

## What It Does

Herd provides everything you need to run Laravel applications locally, with zero configuration.

| Feature | Description |
|---------|-------------|
| **PHP Versions** | Multiple PHP versions, instantly switchable |
| **Nginx** | Pre-configured web server |
| **Database** | Built-in MySQL and PostgreSQL |
| **Services** | Redis, Meilisearch, MinIO included |
| **SSL** | Automatic HTTPS for local sites |
| **Node.js** | Integrated Node version management |

---

## Our Usage

Herd is our **standard local development environment**. All engineers use Herd for:

- Running TC Portal locally
- Testing across PHP versions
- Local database management
- Queue and worker testing
