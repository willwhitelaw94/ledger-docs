[Cloudflare](https://www.cloudflare.com/) provides CDN, DNS management, DDoS protection, and web security services.

---

## What It Does

Cloudflare sits between users and our servers, providing performance and security benefits.

| Feature | Description |
|---------|-------------|
| **CDN** | Global content delivery network |
| **DNS** | Fast, reliable DNS management |
| **DDoS Protection** | Automatic attack mitigation |
| **SSL/TLS** | Free SSL certificates |
| **WAF** | Web application firewall |
| **Caching** | Edge caching for static assets |

---

## Our Usage

We use Cloudflare for:

- DNS management for all domains
- CDN for static asset delivery
- DDoS protection
- SSL certificate management
- Page rules and redirects
