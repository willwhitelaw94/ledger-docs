[TablePlus](https://tableplus.com/) is a modern, native database management tool with a clean interface.

---

## What It Does

TablePlus provides a GUI for working with databases locally and remotely.

| Feature | Description |
|---------|-------------|
| **Multi-Database** | MySQL, PostgreSQL, SQLite, Redis, and more |
| **Query Editor** | Syntax highlighting, autocomplete |
| **SSH Tunneling** | Secure connections to remote databases |
| **Data Editing** | Inline editing with validation |
| **Export/Import** | CSV, JSON, SQL dumps |

---

## Our Usage

We use TablePlus for:

- Local development database management
- Debugging data issues
- Quick queries and data exploration
- Connecting to staging environments (via SSH)

---

## Setup

1. Download from [tableplus.com](https://tableplus.com/)
2. Connect to your local Laravel Herd databases
3. For remote access, use SSH tunneling through approved channels
