It's not just for coding. It's an autonomous agent that can orchestrate complex, multi-step tasks.

## Anthropic's Mission

[Anthropic](https://www.anthropic.com/company) builds AI that's safe, beneficial, and steerable. Founded by former OpenAI researchers, they're focused on AI safety - building systems people can actually rely on.

Claude Code is their agentic coding tool, but "coding" undersells it. It's really about **multi-step task orchestration** with human checkpoints.

## More Than a Coding Tool

People are using Claude Code for:

- **Financial planning** - reading through bank statements, categorizing expenses, building budgets
- **Business planning** - authors mapping out business strategies and plans
- **Research** - analyzing documents, extracting insights, summarizing findings
- **Knowledge work** - any complex task that benefits from AI that can read files, think, and take actions

The "Code" in Claude Code is somewhat misleading. It's an agent that happens to be excellent at code, but the same capabilities work for any knowledge work.

## How It Differs from Copilot & Cursor

| Tool               | Philosophy                | Best For                                     |
| ------------------ | ------------------------- | -------------------------------------------- |
| **GitHub Copilot** | Line-by-line autocomplete | Fast completions while typing                |
| **Cursor**         | IDE-integrated AI         | Multi-file edits in editor                   |
| **Claude Code**    | Autonomous agent          | Complex multi-step tasks, thinking, planning |

Copilot and Cursor augment your typing. Claude Code **thinks and acts** - it plans, executes, verifies, and iterates.

> "Claude doesn't rush to rewrite code. It explains the root cause."

## Where Claude Code Runs

### Terminal (Native)

The original interface. Run `claude` in any terminal and you're in a conversation with full access to your filesystem.

### Cursor / VS Code Extension

Claude Code integrates as an extension. Best of both worlds - IDE convenience with Claude's capabilities.

**My preference**: Multiple Claude Code sessions in Cursor, each working on different parts of the codebase.

### Claude Desktop

The standalone app. Good for non-coding tasks or when you don't need filesystem access.

## Key Takeaways

- Claude Code is **agent orchestration**, not just code completion
- People use it for finance, business planning, research - not just coding
- Runs in terminal, Cursor/VS Code, or Claude Desktop
- Best experience: Cursor with Claude Code extension
- Anthropic's focus on safety means Claude thinks before acting
