PATTERN ANNOTATIONS:
- Tab filters (GitHub: Open/Closed) → Status tabs for quick filtering
- Status badges (Primer State Labels) → Color-coded UNLINKED/LINKED/PENDING
- Search bar (GitHub query syntax) → Smart search with filter keywords
- Checkbox selection (GitHub bulk ops) → Select multiple for batch actions
- Preview line (GitHub PR description) → Transcript snippet for context
- Package link (GitHub repo badge) → Shows linked/unlinked package status

PATTERN ANNOTATIONS:
- Split pane (GitHub: Files + Sidebar) → Transcript left, actions right
- [Viewed] checkbox (GitHub: Mark file as viewed) → Track review progress
- Inline comment [+ 💬] (GitHub: Line comments) → Notes on specific moments
- Three-state submit (GitHub: Comment/Approve/Request Changes):
  - Just Comment = add note without completing
  - Link & Complete = approve and close
  - Needs Attention = flag for escalation
- Audio player (analogous to diff view) → The content being reviewed
- Timestamp navigation → Like clicking line numbers in diff

PATTERN ANNOTATIONS:
- Grouped by repository (GitHub: Notification grouping) → Group by Package
- Saved filters (GitHub: Custom filter presets) → Quick access to common views
- Inline row actions (GitHub: Mark as read/Archive) → Play, Link, Complete in-row
- Collapsible groups (GitHub: Expandable sections) → Focus on one package at a time
- AI suggestions (GitHub Copilot pattern) → Auto-suggest package matches
- Reason-based filtering → Filter by why call needs attention
