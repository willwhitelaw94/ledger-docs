================================================================================

                          KEYBOARD SHORTCUT MAP
================================================================================

All variations share these Superhuman-inspired shortcuts:

  NAVIGATION
  ──────────────────────────────────
  J         │ Next call
  K         │ Previous call
  Enter     │ Expand/focus call
  Esc       │ Collapse/exit focus
  1/2/3     │ Switch inbox tabs

  AUDIO
  ──────────────────────────────────
  Space     │ Play/pause audio
  ←         │ Seek back 10 seconds
  →         │ Seek forward 10 seconds
  [ ]       │ Decrease/increase speed

  ACTIONS
  ──────────────────────────────────
  E         │ Complete review (log billing)
  L         │ Link to package
  N         │ Add note
  H         │ Defer (snooze)

  POWER USER
  ──────────────────────────────────
  Cmd+K     │ Command palette
  ;         │ Insert snippet
  ?         │ Show all shortcuts


================================================================================

                           SNIPPET LIBRARY
================================================================================

Quick note templates (type ; then shortcut):

  ;routine     →  "Routine wellness check. Client confirmed wellbeing."

  ;followup    →  "Follow-up required: [reason]. Assigned to [coordinator]."

  ;escalate    →  "Escalated to supervisor. Concern: [reason]"

  ;noaction    →  "Informational call only. No action required."

  ;appt        →  "Appointment inquiry. Confirmed [date] at [time]."

  ;respite     →  "Respite care discussion. [hours] hours requested for [date]."

  ;feedback    →  "Client feedback: [positive/concern]. Re: [topic]"
