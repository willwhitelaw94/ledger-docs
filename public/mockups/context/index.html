Chip Glossary

Category Name

Shows the AI-detected SERT category. Appears on all AI-matched rows (both sunny day and split).

Split recommended

Only appears when 2+ SERTs detected. Amber warning colour with pulse animation. Clickable — opens the split panel.

Service Name (SERV-xxx)

Shows after user confirms a service. Teal (success) indicates a linked planned service. The end state.

Colour Convention

Purple — AI intelligence (confidence, suggestions)
Amber — Warning / action needed (split recommended)
Teal — Confirmed / linked (user action completed)
